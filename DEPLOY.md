# Deploy checklist

After pushing, enable GitHub Pages once:

1. Open https://github.com/Noahenglmann/noahs-reader/settings/pages
2. **Source:** Deploy from a branch
3. **Branch:** `gh-pages` → `/ (root)` → Save

Live URL: https://noahenglmann.github.io/noahs-reader/

Secret required: `VITE_GOOGLE_API_KEY` in repo Settings → Secrets → Actions
